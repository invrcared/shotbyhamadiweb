# Project Context

**ShotByHamadi Web**

This is the portfolio website for ShotByHamadi, a professional photography and videography business. The website is built using HTML, CSS (with TailwindCSS via CDN), and basic JavaScript.

- **Theme**: Dark mode by default (black background, white text) with a custom sage green accent (`#b4d3b0`).
- **Typography**: Inter font with ultra-wide tracking for headings.
- **Main Goal**: Showcase professional work and provide a clean, premium "get in touch" and "view work" mechanism.
- **Deployment**: Integrated with Cloudflare, building and deploying automatically when changes are pushed to the main branch.
